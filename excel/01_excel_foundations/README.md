# Excel Foundations – Analytical Track

## 🎯 Objective

Build strong Excel skills required for professional Data Analysis,
including modeling, aggregation and dashboard thinking.

---

## 📌 Core Areas

### 1️⃣ Data Cleaning
- Remove duplicates
- Text to columns
- TRIM / CLEAN
- Data validation

### 2️⃣ Essential Formulas
- IF / IFS
- SUMIF / COUNTIF
- INDEX + MATCH
- XLOOKUP
- LEFT / RIGHT / MID

### 3️⃣ Aggregation & Analysis
- Pivot Tables
- Calculated fields
- Grouping (dates, categories)

### 4️⃣ Business Thinking
- KPI calculations
- Revenue metrics
- Growth rates
- Percentage change

---

## 🧠 Goal

Move from “formula user” to analytical model builder.
